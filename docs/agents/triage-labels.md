# Triage Labels

| Canonical role | Label | Meaning |
|---|---|---|
| `needs-triage` | `needs-triage` | Maintainer needs to evaluate the issue |
| `needs-info` | `needs-info` | Waiting on additional information |
| `ready-for-agent` | `ready-for-agent` | Fully specified and ready for implementation |
| `ready-for-human` | `ready-for-human` | Requires human implementation or decision |
| `wontfix` | `wontfix` | Will not be actioned |

When a skill references a canonical triage role, use the corresponding label in the second column.
