# ADR-001：公开的垂直 Agent 项目

| 属性 | 值 |
|---|---|
| 状态 | accepted |
| 日期 | 2026-07-21 |

## 背景

项目目标是申请 Agent 应用开发实习，需要同时证明 Agent 技术深度、业务价值和完整交付能力。

## 决策

建设公开的多模态电商视觉创意 Agent，而不是通用 Agent Builder 或仅供企业采购的超大平台。

## 后果

- 业务范围聚焦电商视觉。
- Agent Runtime、Evaluation 和 Replay 是技术主线。
- 提供在线 Demo、公开评测和可部署代码。
- 企业高可用仍属于最终系统，但不抢占 Agent 核心开发顺序。

## 验证

- HR 能在五分钟内理解产品价值。
- 技术面试可以展示 Planning、Tool、Memory、HITL、Reflection 和 Recovery。
- 公开指标可复现。
